# Consolidador Fiscal B3

Um consolidador de informações fiscais da B3 (Bolsa de Valores Brasileira), desenvolvido em Python para facilitar o processamento e análise de dados de operações.

## 🎯 Objetivo

Este projeto visa consolidar e processar informações fiscais de operações realizadas na B3, facilitando a geração de relatórios e análises necessárias para declaração de impostos e gestão financeira.

## 🚀 Funcionalidades

- [ ] Importação de dados de operações B3
- [ ] Consolidação de informações fiscais
- [ ] Geração de relatórios
- [ ] Análise de dados

## 📋 Pré-requisitos

- Python 3.8+
- pip (gerenciador de pacotes Python)

## 💻 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/vandersonr-souza/consolidador-b3.git
cd consolidador-b3
```

2. Crie um ambiente virtual:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# ou
venv\Scripts\activate  # Windows
```

3. Instale as dependências:
```bash
pip install -r requirements.txt
```
